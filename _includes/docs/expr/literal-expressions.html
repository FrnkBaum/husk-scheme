Literal Expressions

syntax: (quote <datum>) syntax: '<datum> syntax: <constant>
(quote ) evaluates to . may be any external representation of a Scheme object (see section 3.3). This notation is used to include literal constants in Scheme code. {% highlight scheme %} (quote a) ===> a (quote #(a b c)) ===> #(a b c) (quote (+ 1 2)) ===> (+ 1 2) {% endhighlight %} (quote ) may be abbreviated as '. The two notations are equivalent in all respects. {% highlight scheme %} 'a ===> a '#(a b c) ===> #(a b c) '() ===> () '(+ 1 2) ===> (+ 1 2) '(quote a) ===> (quote a) ''a ===> (quote a) {% endhighlight %} Numerical constants, string constants, character constants, and boolean constants evaluate ``to themselves''; they need not be quoted. {% highlight scheme %} '"abc" ===> "abc" "abc" ===> "abc" '145932 ===> 145932 145932 ===> 145932 '#t ===> #t #t ===> #t {% endhighlight %}
