Input/Output

Ports represent input and output devices. To Scheme, an input port is a Scheme object that can deliver characters upon command, while an output port is a Scheme object that can accept characters.

newline

{% scmdoc newline "(newline)" Function %}
(newline port)
Writes an end of line to port. The port argument may be omitted, in which case it defaults to the value returned by current-output-port. {% endscmdoc %}
